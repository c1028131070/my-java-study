## java并发编程基础
#### 1、线程基础
在介绍什么是线程之前，有必要对进程进行了解下，在操作系统中线程是进程中的一个实体，线程并不会独立存在，进程是资源分配和调度的基本单位，一个进程中最少有一个线程，多个线程共享一个进程内的资源。
